Installation

Windows

The easiest way to install on Windows:

  1. Download and extract SAOIS
  2. Double-click INSTALL_ME.bat
  3. Follow the prompts

Or use PowerShell:

cd path\to\SAOISCLI
.\scripts\install.ps1

macOS / Linux

cd path/to/SAOISCLI
./scripts/install.sh
source ~/.zshrc

💡 After installation, run saois start to complete setup.

Commands

SAOIS has just 6 commands to remember:

Command Description
saois start Quick setup wizard - run this first!
saois work <name> Start working on a project with the best AI tool
saois list Show all registered projects
saois add <name> <path> Add a new project to SAOIS
saois tools Check which AI tools are installed
saois help Show help and all commands

Examples

# First time setup
saois start

# Add a project
saois add my-website ~/projects/my-website

# Start working
saois work my-website

# See all projects
saois list

Prompt Templates

SAOIS includes built-in prompt templates to help you work more effectively with AI tools.

Using Templates

Templates are stored in docs/templates/ in your SAOIS folder. You can:

  • Copy and paste them into your AI tool
  • Customize them for your specific needs
  • Create your own templates

Project Brain

Each project can have a "brain" file that tells SAOIS what you're working on. This helps SAOIS pick the right AI tool.

How It Works

  1. SAOIS looks for docs/project_brain.md in your project
  2. It reads the task type (code, research, or plan)
  3. It launches the best AI tool for that task

Task Types

Task Type Best Tools Use For
code Windsurf, Cursor, VS Code Writing code, debugging, features
research Perplexity, ChatGPT Learning, exploring, finding solutions
plan Claude, ChatGPT Architecture, design, planning

Example Project Brain

# My Website - Project Brain

## What is this project?
A personal portfolio website built with Next.js

## Current Status
In development - building the blog section

## What I'm Working On
**Task Type:** code
**Current Task:** Add markdown blog support

## How to Run
```bash
npm run dev
```

💡 If no brain file exists, SAOIS will ask what you're working on and create one automatically!

Troubleshooting

"command not found: saois"

Windows:

  • Close and reopen PowerShell
  • Or run: . $PROFILE
  • Or use: python -m saois.simple_cli help

Mac/Linux:

  • Run: source ~/.zshrc
  • Or close and reopen Terminal

"Python not found"

Install Python 3.9+ from python.org

⚠️ On Windows, make sure to check "Add Python to PATH" during installation!

"No AI tools found"

SAOIS will use browser versions of AI tools if no desktop apps are installed. For the best experience, install:

  • Windsurf - AI-powered coding
  • Cursor - AI-first editor
  • VS Code - General coding

"Project not found"

Make sure the project is registered:

saois list                    # See all projects
saois add myproject ~/path    # Add a project

Still having issues?

Try running the direct Python command:

python -m saois.simple_cli help

If this works, the issue is with your shell configuration. Re-run the installer.
