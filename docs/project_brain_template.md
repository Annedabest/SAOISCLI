## Task Type Guide

- **coding**: Writing new features or functionality
- **debugging**: Fixing bugs or errors
- **architecture**: Designing system structure or making architectural decisions
- **research**: Finding information, comparing solutions, learning new tech
- **analysis**: Understanding existing codebase or analyzing performance
- **planning**: Project planning, roadmap creation, task breakdown
- **automation**: Setting up CI/CD, scripts, or automated workflows
